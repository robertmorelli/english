Technical Details

Compact Bitset Trie

Each node is only 8 bytes using bitmasks instead of pointers.

Super-Letter Compression

Common pairs (th, he, in, er, an, re) compressed to single tokens.

Level-Based Storage

Nodes organized by depth for O(1) child lookup.

~4MB Total Size

236,000 words in a fraction of traditional trie memory.
